# Gabi Project Frontend

Esta pasta contém o frontend separado do backend Rails, feito com Vue.js, Vite, Tailwindcss e Cypress.

## Scripts

- `npm run dev` — inicia o Vite em modo desenvolvimento.
- `npm run build` — gera build de produção.
- `npm run preview` — pré-visualiza o build.
- `npm run cy:open` — abre o Cypress.
- `npm run cy:run` — executa os testes do Cypress.
